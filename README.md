# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website developed using HTML5. The website showcases personal information, technical skills, and contact details in a structured and user-friendly format.

The objective of this project is to practice fundamental HTML concepts such as semantic tags, forms, navigation, images, lists, and hyperlinks.

---

## Features

* Personal introduction section
* Skills section
* Contact form with input validation
* Navigation menu with internal links
* Profile image
* External links to professional profiles
* Semantic HTML structure

---

## Technologies Used

* HTML5
